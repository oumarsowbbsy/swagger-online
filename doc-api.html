Utilisateur ¶

Inscription  ¶

Cette action permet d’inscrire un utilisateur. Alias de /php5/restapi/utilisateur/register

Inscription utilisateur
POST/php5/restapi/utilisateur/inscription

Example URI

POST dev-pf.onlineformapro.com/php5/restapi/utilisateur/inscription
URI Parameters
HideShow
nom
string (required) Example: DUPONT

Le nom du nouvel utilisateur

prenom
string (required) Example: Jean

Le prénom du nouvel utilisateur

login
string (optional) Example: j.dupont

Le login de l’utilisateur.

pwd
string (optional) Example: gtxdsG45!df$aez

Le mot de passe de l’utilisateur. Si non spécifié, le mot de passe est généré aléatoirement

email
string (optional) Example: j.dupont@mail.com

L’email du nouvel utilisateur. Il n’est pas possible de créer plusieurs comptes avec le même email.

civilite
number (optional) Example: 2

La civilité du nouvel utilisateur 1 pour Monsieur, 2 pour Madame. La valeur par défaut est 1

date_debut
string (optional) Example: 2021-04-01

La date de début de validité du compte utilisateur au format YYYY-mm-dd

date_fin
string (optional) Example: 2021-10-31

La date de fin de validité du compte utilisateur au format YYYY-mm-dd

valide
string (optional) 

1 pour rendre l’utilisateur valide, 0 pour le rendre invalide. Un utilisateur invalide ne peut plus se connecter

groupes
string (optional) 

La liste des libellés des groupes dans lesquels sera l’utilisateur.

profil_formation
string (optional) 

Le libellé du profil de formation dans lequel sera l’utilisateur.

profils_formation
string (optional) 

La liste des libellés des profils de formation dans lesquels sera l’utilisateur.

profil_evaluation
string (optional) 

Le libellé du profil d’évaluation dans lequel sera l’utilisateur.

profils_evaluation
string (optional) 

La liste des libellés des profils d’évaluation dans lesquels sera l’utilisateur.

profil_parcours
string (optional) 

Le libellé du profil de parcours dans lequel sera l’utilisateur.

profils_parcours
string (optional) 

La liste des libellés des profils de parcours dans lesquels sera l’utilisateur.

envoi_notification
string (optional) 

1 pour qu’une notification par email soit envoyé au nouvel utilisateur. 0 sinon.

Affect formations  ¶

Cette action permet d’affecter des formations à un utilisateur

Affecter des formations
POST/php5/restapi/catalogue/affect-formations

Example URI

POST dev-pf.onlineformapro.com/php5/restapi/catalogue/affect-formations
URI Parameters
HideShow
utilisateur
string (required) Example: 2F81B1853DD87BD88E7BC3E465B59271D149

Le GUID de l’utilisateur

modules
string (optional) Example: [14725, 13258]

La liste des ids des modules à affecter

dates
string (optional) Example: {"14725":{"date_debut":"2021-02-01", "date_fin":"2021-10-01"}}

Les limites par id modules des dates pendant laquelle l’utilisateur pourra utiliser le sous-thème dans lequel est le module

durees
string (optional) Example: {"14725":2, "13258":5}

La durée maximum par id module pendant laquelle l’utilisateur peut utiliser le sous-thème dans lequel est le module (en heures)

sousthemes
string (optional) Example: [24316, 34215]

La liste des ids des sous-thèmes à affecter

dates_sousthemes
string (optional) 

Les limites par id sous-thème des dates pendant laquelle l’utilisateur pourra utiliser le sous-thème. Même syntaxe que dates

durees_sousthemes
string (optional) Example: {"24316":10}

Les durées maximum par id sous-thème pendant laquelle l’utilisateur peut utiliser le sous-thème (en heures)

themes
string (optional) Example: [19458, 14753]

La liste des ids des thèmes à affecter

dates_themes
string (optional) 

Les limites par id thème des dates pendant laquelle l’utilisateur pourra utiliser le thème. Même syntaxe que dates.

durees_themes
string (optional) Example: {"19458":20}

Les durées maximum par id thème pendant laquelle l’utilisateur peut utiliser le thème (en heures)

tutorat
string (optional) Example: 1

1 pour activer le tutorat, 0 sinon

affect_eval_pre
string (required) Example: 1

1 pour affecter les évaluations pre.

affect_eval_post
string (required) Example: 1

1 pour affecter les évaluations post.
